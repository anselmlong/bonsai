Cluster 2 — Key Signals from Dev Communities

⚠️ Tavily Acquired by Nebius (Feb 2026)

Amsterdam-based AI cloud company Nebius acquired Tavily. Some devs are evaluating alternatives over concern about product direction changes. Tavily still works great today — but it's a risk flag for long-term projects.

🔥 Exa Outperforms on Semantic Search

81% vs 71% on complex multi-hop retrieval. 2–3x faster than Tavily. Also supports people, company, and code search — useful for research agents going deep on a topic.

✅ Tavily Still Has Best LangGraph DX

First-class LangChain/LangGraph tool integration. Cleanest result schema for agent consumption. Best documentation for the exact use case. Free tier (1k credits/mo) covers prototyping comfortably.

Cluster 3 — Consumer AI Search (Perplexity-style inspiration)

Relevant as design inspiration for your frontend node UI

Perplexity

94.3% citation accuracy. Deep Research does dozens of searches → hundreds of sources → comprehensive report. Now uses Claude Opus 4.6. 45M MAU. Best UX inspiration for your node graph.

ChatGPT Search

Fastest-growing. Conversational + search combined. Less transparent sourcing than Perplexity. Holds ~86% market share combined with Gemini.

Grok

Real-time X/Twitter data. Fast and conversational. Best for trending topics, weak for deep academic research.

Recommendation for Bonsai

Start with Tavily — best LangGraph DX, cleanest agent-native results, free tier covers dev. Design the search tool interface abstractly so you can swap to Exa or Brave later. The Nebius acquisition is worth watching but doesn't affect you today. For the frontend, Perplexity's source panel is your best design reference.
