Key architectural decisions
asyncio.Queue per job
Graph runs in background task. Every node event is pushed to a queue. SSE endpoint drains the queue. No polling, no DB needed.
Typed NodeEvent schema
Every event carries node_id, parent_id, depth, type. Frontend reconstructs the tree purely from the event stream.
BranchGraph is isolated
Takes a question + depth + config. Returns sources + summary. Fully testable in isolation without the root graph.
Dive Deeper endpoint
POST /research/:id/dive-deeper with a node_id spawns a new BranchGraph rooted at that node. Same SSE stream, new events appended.
