# scripts/eval.py
# Usage: python scripts/eval.py --n 50 --output results/eval-2026-04-17.json

# Pipeline:
#  1. Load N questions from SimpleQA dataset (huggingface or local CSV)
#  2. Run each through the research agent (async, configurable concurrency)
#  3. Grade each answer with LLM-as-judge (5 dimensions)
#  4. Write per-question JSONL + summary report

class EvalResult(TypedDict):
    question:    str
    gold_answer: str
    agent_answer: str
    latency_s:   float
    branch_count: int
    source_count: int
    scores: dict[str, float]   # per-dimension 0.0вЂ“1.0

# Grading dimensions (LLM-as-judge, single prompt per question):
DIMENSIONS = [
    "factual_accuracy",    # does the answer match gold?
    "citation_accuracy",   # are sources real and relevant?
    "completeness",        # does it cover all aspects of the question?
    "source_quality",      # authoritative sources vs SEO content?
    "conciseness",         # no hallucinated padding?
]

Testing strategy вЂ” what gets tested and how
Unit: individual nodes
test_nodes.py вЂ” each node function takes state in, returns state out. Mock the LLM and Tavily. Fast, deterministic.
Unit: BranchGraph
test_branch_graph.py вЂ” run the full BranchGraph with mocked search + LLM. Tests recursion logic, depth capping, event emission.
Integration: full graph
test_research_graph.py вЂ” marked @pytest.mark.slow. Real Tavily + LLM. Run manually or in CI with --run-slow flag.
Eval: SimpleQA
scripts/eval.py вЂ” separate from pytest. Outputs a JSON report you can include in the README as a benchmark result.
